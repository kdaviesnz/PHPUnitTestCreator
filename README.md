# PHPUnitTestCreator

Create unit tests for classes.

Current restrictions:
- Only works with pure methods.
- Class constructor must have no parameters.

(It is beta!)


## Install

Via Composer

``` bash
$ composer require kdaviesnz/PHPUnitTestCreator
```

## Usage

See test.php file.
